# 🧬 DNA Mutation Analysis Platform

A full-stack AI-powered web application that predicts the clinical significance of DNA mutations using Machine Learning.

## 🚀 Live Demo

Frontend:
https://dna-mutation-analysis-platform.vercel.app/

Backend API:
https://dna-mutation-backend-37fm.onrender.com

---

## 📌 Features

- User Registration & Login Authentication
- DNA Mutation Prediction
- Confidence Score Display
- PDF Report Generation
- Mutation Type Selection
- Gene Symbol Selection
- Prediction History
- Responsive Modern UI
- FastAPI Backend
- React + Vite Frontend
- Machine Learning Integration

---
